## Overview

ApexxCloud provides powerful cloud storage and media transformation capabilities. This guide will help you get started with:

- Setting up your SDK
- Uploading your first file
- Applying transformations

## Installation

Choose the SDK that best fits your needs:

<CodeGroup>

```bash npm
npm install @apexxcloud/sdk-node    # For Node.js
npm install @apexxcloud/sdk-js      # For browser
npm install @apexxcloud/react       # For React
```

```bash yarn
yarn add @apexxcloud/sdk-node      # For Node.js
yarn add @apexxcloud/sdk-js        # For browser
yarn add @apexxcloud/react         # For React
```

</CodeGroup>

## Quick Setup

### Node.js

```javascript
const ApexxCloud = require("@apexxcloud/sdk-node");

const storage = new ApexxCloud({
  accessKey: "your-access-key",
  secretKey: "your-secret-key",
  region: "APAC",
  bucket: "default-bucket",
});
```

### Browser

```javascript
import ApexxCloud from "@apexxcloud/sdk-js";

const storage = new ApexxCloud({
  baseUrl: "https://api.apexxcloud.com",
});
```

### React

```jsx
import { FileUploader } from "@apexxcloud/react";

function App() {
  const getSignedUrl = async (file) => {
    // Get signed URL from your backend
    const response = await fetch("/api/get-signed-url");
    return response.json();
  };

  return (
    <FileUploader
      getSignedUrl={getSignedUrl}
      onUploadComplete={(response) => console.log("Upload complete:", response)}
    />
  );
}
```

## Basic Operations

### Upload a File

<CodeGroup>

```javascript Node.js
const result = await storage.files.upload("my-bucket", "./path/to/image.jpg", {
  visibility: "public",
});
```

```javascript Browser
const fileInput = document.querySelector('input[type="file"]');
const file = fileInput.files[0];

await storage.files.upload(signedUrl, file, {
  onProgress: (event) => console.log(`Progress: ${event.progress}%`),
});
```

</CodeGroup>

### Apply Transformations

Once your file is uploaded, you can apply various transformations using URL parameters:

#### Image Transformations

```
https://cdn.apexxcloud.com/f/access_id/tr-w_500,h_300,c_fill/image.jpg
```

<AccordionGroup>
  <Accordion icon="image" title="Common Image Transformations">
    - Resize: `tr-w_500,h_300`
    - Crop: `tr-c_crop,g_face`
    - Format conversion: `tr-f_webp`
    - Quality: `tr-q_auto:good`
  </Accordion>
  
  <Accordion icon="video" title="Common Video Transformations">
    - Resize: `tr-w_1280,h_720`
    - Thumbnail: `thumb-w_100,h_100`
    - Format: `tr-f_mp4`
  </Accordion>

  <Accordion icon="file" title="Common Document Transformations">
    - Convert to PDF: `tr-f_pdf`
    - Extract thumbnail: `thumb-w_100,h_100`
    - Compress: `tr-c_high`
  </Accordion>
</AccordionGroup>
